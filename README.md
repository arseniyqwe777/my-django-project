# 📚 BookBridge — Книжный мост

Библиотечная система на Django с поиском, авторизацией и админ-панелью.

## Технологии
- Python 3.11
- Django 5.1
- PostgreSQL
- Docker / Gunicorn
- HTML / CSS / Bootstrap

## Функционал
- CRUD для книг, авторов, произведений
- Поиск по названию и автору
- Расширенный поиск с фильтрацией
- Регистрация и авторизация
- Админ-панель
